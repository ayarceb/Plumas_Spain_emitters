# CSV_2021_TOP (datos de emisiones NO₂)

Esta carpeta incluye series de ejemplo para cada planta con el formato
`<PLANTA>_NO2_timeseries_2km_Ktyear.csv` (hora a hora desde 2021-01-01).

Si dispones de los ficheros oficiales, sustitúyelos manteniendo los nombres para
que `app.js` los detecte automáticamente tanto desde `data/` como desde la raíz
del servidor estático.
